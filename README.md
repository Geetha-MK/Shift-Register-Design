# 🔷 Shift Register Design in Verilog

## 📌 Project Overview

This project implements a **4-bit Shift Register** using Verilog HDL. It demonstrates data storage and shifting operations used in digital systems.

---

## 🎯 Features

* Shift Left and Shift Right operations
* Parallel Data Load
* Serial Input support
* SIPO and PISO functionality

---

## 🛠️ Tools Used

* Verilog HDL
* EDA Playground
* ModelSim

---

## 📁 Files Included

* `shift_register.v` – Design module
* `testbench.v` – Simulation testbench

---

## ▶️ How to Run

1. Open EDA Playground
2. Paste design and testbench
3. Run simulation
4. View waveform

---

## 📊 Output

* Data is shifted left and right correctly
* Parallel load works as expected
* Verified using waveform
